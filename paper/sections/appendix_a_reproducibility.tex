\section{Reproducibility Manifest}
\label{sec:appendix_repro}

\subsection{Software and Artifacts}
\begin{itemize}[leftmargin=*]
  \item MICM code source and commit hash.
  \item MICM archived software DOI.
  \item Build system, compiler, and flags.
\end{itemize}

\subsection{Runtime Environment}
\begin{itemize}[leftmargin=*]
  \item CPU model, core count, memory.
  \item Operating system and kernel.
  \item Threading or vectorization settings relevant to solver runs.
\end{itemize}

\subsection{Data and Plots}
\begin{itemize}[leftmargin=*]
  \item Raw benchmark outputs (CSV/JSON).
  \item Reported columns include $c$-error norms, $\|g(c_n)\|$ residual norms, step counts, and linear solves.
  \item Plot-generation scripts and exact command lines.
  \item Hashes for released result bundles.
\end{itemize}
